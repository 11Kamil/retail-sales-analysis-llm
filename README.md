# Retail Sales Analysis Using LangChain and LLMs

## Project Overview

This project applies exploratory data analysis, machine learning clustering, and statistical hypothesis testing to a large-scale retail sales dataset using LangChain and Large Language Models (LLMs).

The analysis was performed on a retail dataset containing over 421,000 sales records and focused on uncovering operational, seasonal, and economic insights affecting retail performance.

## Objectives

* Perform Exploratory Data Analysis (EDA)
* Analyze retail sales trends
* Identify high-performing and low-performing store groups
* Apply machine learning clustering techniques
* Conduct statistical hypothesis testing
* Generate business insights using LLM-powered workflows

## Technologies Used

* Python
* Pandas
* LangChain
* OpenAI API
* Scikit-learn
* Statsmodels
* Matplotlib
* Seaborn

## Key Findings

* Holiday periods significantly increase weekly sales
* Monthly sales patterns show strong statistical significance
* Higher unemployment levels negatively impact retail sales
* Store clusters exhibit different operational and economic behaviors
* Store Type A locations generated the highest sales performance

## Machine Learning & Statistical Methods

* K-Means Clustering
* ANOVA Testing
* Linear Regression Analysis
* Correlation Analysis
